# outputs/

Generated artifacts (git-ignored except this README):

```
tables/       metric tables (CSV/JSON)
figures/      plots (layer sweeps, heatmaps, coverage-risk, reliability)
activations/  cached hidden states (large; never committed)
checkpoints/  fitted probes / calibrators
reports/      scientific report (methods, results, limitations)
```

Data, activations and results are kept separate (spec section 17). Nothing here
should contain patient-identifiable information.
